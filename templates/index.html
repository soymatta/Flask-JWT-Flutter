Flask Backend Project with SQLAlchemy and JWT 🚀

This project implements a Flask backend with SQLAlchemy for managing a PostgreSQL database and JWT authentication.

JWT guide 📜

This is the guide about how to get JTW for the authenticated requests:

  1. If you don't know any user and password, you have to register a new user
    • https://flask-jwt-flutter.onrender.com/api/users/register
      📝 | POST (Register a new user)
    • https://flask-jwt-flutter.onrender.com/api/nutritionists/register
      📝 | POST (Register a new nutritionist)
  2. If you have a user and password, you have to login to get the JWT
    • https://flask-jwt-flutter.onrender.com/api/login
      🔑 | POST (User or Nutritionist login)

Endpoints 📡

Below are the available endpoints:

  • https://flask-jwt-flutter.onrender.com/api/users
    👤 | GET (Get all users), POST (Create a new user)
  • https://flask-jwt-flutter.onrender.com/api/users/<user_id>
    👤 | GET (Get a specific user), PUT (Update a specific user), DELETE (Delete a specific user)
  • https://flask-jwt-flutter.onrender.com/api/nutritionists
    🥦 | GET (Get all nutritionists), POST (Create a new nutritionist)
  • https://flask-jwt-flutter.onrender.com/api/nutritionists/<nutritionist_id>
    🥦 | GET (Get a specific nutritionist), PUT (Update a specific nutritionist), DELETE (Delete a specific nutritionist)
  • https://flask-jwt-flutter.onrender.com/api/comments
    💬 | GET (Get all comments), POST (Create a new comment)
  • https://flask-jwt-flutter.onrender.com/api/comments/<comment_id>
    💬 | GET (Get a specific comment), PUT (Update a specific comment), DELETE (Delete a specific comment)
  • https://flask-jwt-flutter.onrender.com/api/professional_tips
    💼 | GET (Get all professional tips), POST (Create a new professional tip)
  • https://flask-jwt-flutter.onrender.com/api/professional_tips/<professional_tip_id>
    💼 | GET (Get a specific professional tip), PUT (Update a specific professional tip), DELETE (Delete a specific professional tip)
